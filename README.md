**# Workout Tracker**



[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)



**## Description**

This is a web app that allows you to view, create, and track daily workouts. 



It uses:

* HTML
* CSS
* Node
* Express
* MongoDB
* MongoDB Atlas
* Mongoose
* JavaScript
* Heroku
